Dáta a AJAX

Tabuľka načítaná z JSON súboru cez AJAX

Dáta spoločnosti

Nižšie je zobrazená tabuľka so štruktúrovanými dátami načítanými z JSON súboru. Dáta obsahujú hierarchickú štruktúru s aspoň 3 úrovňami vnorenia (spoločnosť → zamestnanci → projekty).

Kliknite na tlačidlo "Načítať dáta" pre zobrazenie tabuľky cez AJAX.

Technické detaily

  • Dátový zdroj: JSON súbor (data/data.json)
  • Metóda: Fetch API s AJAX
  • Štruktúra: Spoločnosť → Zamestnanci → Projekty
  • Funkcionalita: Dynamické vykresľovanie tabuľky
