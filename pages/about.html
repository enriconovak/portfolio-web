O mne

Kto som?

Som nadšený vývojár webových aplikácií so skúsenosťami v HTML5, CSS3 a JavaScript. Moja vášňa je vytvárať krásne a funkčné webové stránky, ktoré poskytujú skvelý užívateľský zážitok.

Moje zručnosti

  • HTML5 a sémantické značky
  • CSS3 - Flexbox, Grid, Animácie
  • JavaScript - DOM manipulácia, AJAX, Validácia
  • Responzívny dizajn
  • Git a verziovanie
  • JSON dáta a ich spracovanie

Môj prístup

Verím na rovnováhu medzi estetickým dizajnom a funkcionalitou. Každý projekt som snažím prispôsobiť potrebám a očakávaniam klienta. Používam najlepšie praktiky webového vývoja a stále sa učím novým technológiám.
